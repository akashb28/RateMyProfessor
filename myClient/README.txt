MYCLIENT INSTRUCTIONS-

The myClient application is a menu driven command line program. It accepts
the commands 'register', 'login' and 'exit'. Upon registering/logging in
the user will be prompted to enter details such as username,password and email.
Once successfully logged in, the user can use the commmands- 'list','view',
'average', 'rate','logout'. For the average and rate commands, the user 
will be prompted to enter details such as professor id, module id etc.
Upon entering 'logout' the user will be logged out and taken to the
previous menu.
